¿Que es HTML?

HTML, sigla en inglés de HyperText Markup Language (lenguaje de marcas de hipertexto), hace referencia al lemguaje de marcado para la elaboracion de paginas web. Es un estandar que sirve de referencia del software que conecta con la elaboracion de paginas web en sus diferentes versiones, define una estructura basica y un codigo (denomiado codigo HTML) para la definicion de contenido de una pagina web, como texto, imagenes, videos, juegos, entre otros.
